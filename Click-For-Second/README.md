做这个游戏只是为了好玩，  
因使用不当造成的事故本人不承担任何责任。

# 帮助

关于秒数(s)(相当于货币)：

1. 点击工作区即可；  
2. 商店内物品需要一定**研究**以解锁，解锁后即可用**秒数**购买。  

关于研究：

1. 点击研究项目(***此时简介栏会有关于研究的介绍*** )；  
2. 点击“研究”按钮；  
3. 如果秒数足够则开始研究；  
4. 研究需要时间，请**保持耐心**！

关于物品合成：

1. 合成需要物品
2. 有概率**合成失败**

关于随机要素

1. 每过***86400秒(1天)*** 就有**概率**会触发
2. 请为触发后会发生的事***做好心理准备***
