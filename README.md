# 帮助

关于秒数(s)(相当于货币)：

1.点击工作区即可；  
2.商店内物品需要一定**研究**以解锁，解锁后即可用*秒数* 购买。

关于研究：

1.点击研究项目(***此时简介栏会有关于研究的介绍*** )；  
2.点击“研究”按钮；  
3.如果*秒数* 足够则开始研究；   
4.研究需要**时间**，请*保持耐心*！
